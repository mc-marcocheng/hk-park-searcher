COMMUNITY CONTRIBUTION

提交新的街健公園

提交後，系統會建立一個公開 GitHub Pull Request。 資料經管理員審核及合併後才會顯示在公園地圖。

01 公園資料

2–100 個字元。

選填，最多 120 個字元。

3–200 個字元。

選填，最多 240 個字元。

02 公園位置

在下方地圖上點擊以選擇位置，或使用下方的經緯度輸入框。

03 公園環境照片

請上傳能看到公園整體環境的照片，例如入口、全景、周邊環境等。

04 器材記錄

請在對應的欄位上傳該公園內有的器材照片。沒有的器材請直接跳過。

所有欄位合計最多 8 張照片，每張原始檔最多 12 MB。

05 備註及確認

任何補充資料，例如：開放時間、器材狀況、其他器材名稱、前往方式等。

公園評分 *

請根據你的整體訓練體驗評分，可考慮器材種類、器材質素、場地空間、地面狀況等因素。

BEFORE SUBMITTING

投稿須知

  • 只提交可公開的公園資料。
  • 不要填寫電話、電郵或其他私人資料。
  • 只上載由你拍攝或獲授權使用的相片。
  • Pull Request 及提交內容會公開顯示。
