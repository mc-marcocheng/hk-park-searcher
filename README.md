# Hong Kong Park Searcher 香港街健公園搜尋器

![Park Searcher Preview](assets/preview.png)

係一個幫你搵香港街頭健身公園嘅網頁。如果你想搵附近邊度有單槓、雙槓或者其他健身設施，呢度就啱晒你。

A web application to find street workout parks in Hong Kong.
